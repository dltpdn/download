SmartHome Hub 사용자 가이드
버전: 1.0
작성일: 2025-01-01

────────────────────────────────────
1. 제품 개요
────────────────────────────────────
SmartHome Hub(이하 ‘허브’)는 가정 내 IoT 기기를 통합적으로 제어하기 위한 중앙 관리 장치입니다.
Wi-Fi, Bluetooth, Zigbee, Matter 프로토콜을 지원하며 스마트 조명, 도어락, 온도 조절기 등
다양한 기기와 연동됩니다.

주요 특징:
- 음성 인식 기반 제어 지원
- 기기 자동화 시나리오 생성 가능
- 원격 제어 및 상태 모니터링
- 보안 강화를 위한 기기 인증 기능 제공

────────────────────────────────────
2. 구성품
────────────────────────────────────
- SmartHome Hub 본체
- 전원 어댑터
- 이더넷 케이블(1m)
- 간단 설치 매뉴얼

────────────────────────────────────
3. 설치 방법
────────────────────────────────────
3.1 기본 설치 절차
1) 허브를 전원 어댑터에 연결합니다.
2) 전원이 켜지면 전면 LED가 흰색으로 점등됩니다.
3) 허브를 Wi-Fi 또는 이더넷 케이블로 인터넷에 연결합니다.
4) 스마트폰에서 SmartHome 앱을 다운로드하고 계정을 생성합니다.
5) 앱에서 '새 허브 추가'를 선택하여 허브와 스마트폰을 페어링합니다.

3.2 Wi-Fi 연결 방법
- 기본 메뉴 > 네트워크 설정 > Wi-Fi 연결
- 사용 가능한 Wi-Fi 목록에서 네트워크 선택 후 비밀번호 입력
- 연결 성공 시 LED가 파란색으로 점등됩니다.

3.3 기기 추가 방법
- 앱 홈 화면 > ‘기기 추가’
- 허브에서 Bluetooth 및 Zigbee 검색을 자동 실행
- 등록 가능한 기기 목록이 나타나면 선택 후 ‘추가’를 누릅니다.

────────────────────────────────────
4. 주요 기능 안내
────────────────────────────────────
4.1 음성 제어
- SmartHome Hub는 자체 음성 인식 엔진을 제공하며 다음 명령을 지원합니다.
  예) “거실 불 꺼줘”, “온도 23도로 설정해줘”

4.2 자동화 시나리오
- 조건(온도, 시간, 움직임 등)에 따라 동작을 자동 실행할 수 있습니다.
  예) “오후 7시에 거실 조명 자동 켜기”

4.3 보안 기능
- 기기 등록 시 고유 식별 번호 기반 인증 수행
- 원격 접속 시 2단계 보안 코드 입력 요구

────────────────────────────────────
5. LED 상태 표시
────────────────────────────────────
- 흰색: 전원 정상, 초기화 상태
- 파란색: Wi-Fi 연결 성공
- 노란색: 업데이트 진행 중
- 빨간색: 네트워크 오류

────────────────────────────────────
6. 문제 해결(FAQ)
────────────────────────────────────
Q1. 허브가 Wi-Fi에 연결되지 않습니다. 어떻게 해야 하나요?
A1.
- Wi-Fi 비밀번호가 정확한지 확인하세요.
- 공유기에 2.4GHz 대역이 활성화되어 있는지 확인하세요.
- 허브를 재부팅한 후 다시 연결을 시도하세요.
- 그래도 연결되지 않으면 ‘네트워크 초기화’를 실행하세요.
  (설정 > 시스템 > 네트워크 초기화)

Q2. 기기가 검색되지 않아요.
A2.
- 기기가 페어링 모드인지 확인하세요.
- Bluetooth 또는 Zigbee 연결 범위 내에 허브가 있는지 체크하세요.
- 이미 다른 허브에 등록된 기기는 재등록이 필요합니다.

Q3. LED가 계속 빨간색입니다.
A3.
- 인터넷 연결 상태를 확인하세요.
- 이더넷 케이블이 올바르게 연결되었는지 점검하세요.
- 라우터 재부팅 후에도 문제가 지속되면 고객 지원팀에 문의하세요.
